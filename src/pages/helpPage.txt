Hentai Terminal: помощь по командам.
Все команды, представленные в Hentai Terminal:

HELP              Вывести информацию по командам.
CLEAR             Очистить экран от вывода результатов выполнения.
ERRORTEST         Тестирование вывода ошибки.
GOODTEST          Тестирование вывода успешного выполнения.
DOCUMENTATION     Просмотреть документацию по Hentai Terminal.
OUTPUTTEST        Тестирование вывода консоли.
CALCULATOR        Запуск апплета: Калькулятор.
LOGOUT            Выйти из учётной записи.
EXIT              Выйти из терминала.
VERSION           Вывод информации о версии и сборке программы.
